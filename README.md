# Verilog Practice

Learning digital design and hardware description languages.

## About

This repository contains Verilog modules I've written while learning digital design fundamentals. All modules have been simulated and tested in Xilinx Vivado.

## Contents

### Combinational Circuits
- Full adder (built from half adders)
- 4 bit Magnitude comparator
- 2-to-1 multiplexer 
- 4 bit Ripple Carry Adder

### Sequential Circuits
- D flip-flop with reset
- Up/down counter

### Finite State Machines
- Traffic light controller
- 3-state FSM with timing control

## Tools Used

- **HDL:** Verilog
- **Simulation:** Xilinx Vivado Behavioral Simulation
- **Editor:** Notepad++
